Build Environment for Ubuntu

Build Environment includes install software as below:

  • update system
  • install chrome
  • install curl
  • install node
  • install Git and configure
  • install vscode
  • install fcitx
  • install fcitx-googlepinyin
  • Set shortcuts

Update systemâ€‹

sudo apt-get update
sudo apt upgrade

Install Chromeâ€‹

Use wget to download chrome deb package:

 wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb

After this finish run the below code:

 sudo dpkg -i google-chrome*
 sudo apt-get -f install

Then we can open chrome with this command:

 /usr/bin/google-chrome-stable

And for convenience we can add chrome process to favorites.

Install curlâ€‹

 sudo apt-get install curl

Install NodeJSâ€‹

First we should set curl source to install the latest node version

 curl -sL https://deb.nodesource.com/setup_16.x | sudo -E bash -

Tip: If you want to install another version of nodejs,you can change the number included in the url on above

Then install nodejs:

  sudo apt-get install -y nodejs

Check node version:

 node -v
 npm -v

Install Git and configure gitâ€‹

 sudo apt-get install git

Config our git for pulling and push code,accessing repository

  1. Generate sshkey
 ssh-keygen -t rsa -C "our email"
 git config --global user.name "our name"
 git config --global user.email "our email"

Tip: If you don't need some other config , when generating sshkey ,we only press enter until generating finished.

Copy the rsa.pub to the environment where our repository was located.

settings -> SSH and GPG keys -> new SSH key

Install vscodeâ€‹

  1. We can search vscode from ubuntu software center.

  1. Downloading vscode deb package.

click here download vscode deb

After this finished, switch to Downloads directory and install it.

cd /home/Downloads
sudo dpkg -g code_*

Open vscode with:

 code .

Install fcitx(for googlepinyin)â€‹

sudo apt-get install fcitx

Occured an error, and we can install fcitx according error message:

sudo apt --fix-broken install

Visit googlepinyin download page

and install it:

  sudo dpkg -i sogoupinyin_*

We need to set system language and restart our computer.

Set shortcutsâ€‹

settings -> keyboard shortcuts -> Screenshots -> Save a screenshot of an area to Pictures
